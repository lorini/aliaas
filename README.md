ALiaaS
======

Description
-----------

ALiaaS is an home automation project. It will considerate each house's element as a service provider. 

Technologies
------------

At this point, ALiaaS uses : 
 - HTML5/CSS UI based on twitter bootstrap/jQuery/jQuery-UI 
 - Node.js observator
 - C++ linux daemon
 - PHP Proxy
